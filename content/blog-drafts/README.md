# How to publish these drafts in Sanity

## Before publishing

Make sure these exist in Sanity:

- author: `KimiClaw Editorial`
- category: `AI Websites`
- category: `Small Business Websites`
- category: `Homepage Design`
- category: `Conversion`
- category: `Sanity CMS`
- category: `Agency Systems`

## For each draft

1. Open the matching markdown file in this folder
2. Copy the title into the `title` field
3. Generate or paste the slug from the file
4. Paste the excerpt into `excerpt`
5. Set the suggested reading time
6. Assign the author
7. Assign the suggested categories
8. Paste the article body into the `body` field
9. Add `seoTitle`
10. Add `seoDescription`
11. Set `publishedAt`
12. Publish

## Recommended publishing order

1. `01-ai-website-design-small-businesses-2026.md`
2. `02-high-converting-service-business-homepage.md`
3. `03-why-sanity-cms-smart-choice-small-agencies.md`

## Important

After publishing:

1. open the live post URL
2. confirm it loads at `/blog/<slug>`
3. resubmit `https://kimiclaw.in/sitemap.xml` in Google Search Console
4. submit the sitemap in Bing Webmaster Tools
